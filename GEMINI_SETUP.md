# 🤖 Gemini AI кілтін қосу (ImmersionChat үшін)

ImmersionChat — AI тәлімгер — нақты жұмыс істеуі үшін Gemini API кілті керек.
Бұл **тегін**. Төмендегі қадамдарды орындаңыз.

---

## 1-қадам: Gemini кілтін алу (тегін)

1. Браузерден ашыңыз: **https://aistudio.google.com/app/apikey**
2. Google аккаунтыңызбен кіріңіз
3. **"Create API key"** (Кілт жасау) батырмасын басыңыз
4. **"Create API key in new project"** таңдаңыз
5. Кілт жасалады — оны **көшіріңіз** (мысалы: `AIzaSyABC123...`)

> ⚠️ Кілтті ешкімге бермеңіз, ашық жерге қоймаңыз.

---

## 2-қадам: Кілтті жобаға қосу

Жоба қалтасында (`app/` ішінде):

1. `.env.example` файлын тауып, оны **көшіріп**, атын `.env` деп өзгертіңіз
   (немесе жаңа `.env` файлын жасаңыз)

2. `.env` файлын ашып, мынаны жазыңыз:
   ```
   VITE_GEMINI_API_KEY=AIzaSyABC123...
   ```
   (`AIzaSyABC123...` орнына өзіңіздің нақты кілтіңізді қойыңыз)

3. Файлды сақтаңыз

---

## 3-қадам: Серверді қайта іске қосу

Кілт қосылғаннан кейін, дев серверді **қайта** іске қосыңыз:

```bash
# Тоқтату: Ctrl+C
# Қайта қосу:
npm run dev
```

> Vite кілттерді тек іске қосылғанда оқиды, сондықтан қайта қосу **міндетті**.

---

## ✅ Тексеру

1. Сайтта **ImmersionChat** бөліміне кіріңіз
2. Енді нұсқаулық емес, **чат** көрінуі керек
3. Ағылшынша бірдеңе жазып көріңіз, мысалы:
   `I go to school yesterday`
4. AI қатені түзетіп, түсіндіруі керек:
   `✓ Correction: I went to school yesterday`
   `• Past tense: use "went"`

---

## 💡 Gemini тегін лимиті

Gemini-дің тегін деңгейі жеткілікті:
- Минутына ~15 сұраныс
- Күніне ~1500 сұраныс

Жеке қолдану үшін бұл әбден жеткілікті.

---

## 🚀 Vercel-ге шығарғанда

Сайтты Vercel-ге деплой жасағанда, кілтті **Vercel баптауында** қосыңыз:

1. Vercel жобаңызда: **Settings → Environment Variables**
2. Қосыңыз:
   - Name: `VITE_GEMINI_API_KEY`
   - Value: сіздің кілтіңіз
3. **Save** → жобаны қайта деплой жасаңыз (Redeploy)

> `.env` файлы GitHub-қа жүктелмейді (қауіпсіздік үшін),
> сондықтан Vercel-ге кілтті бөлек қосу керек.
